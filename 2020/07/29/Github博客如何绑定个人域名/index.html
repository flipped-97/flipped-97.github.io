Github pages博客如何绑定个人域名

2020-07-29
-
Hexo是一个开源的静态博客框架，支持markdown文档，应用起来很方便而且有丰富的主题模板可供选择，是建立个性化的个人博客的不二之选。以下以Github+hexo为例搭建个人博客。

一、购买域名，实名认证，不需要备案

二、域名解析（本地与服务器映射）

  • 在域名购买商控制面板添加解析（DNS管理）

三、接下来会让你输入IP，完成解析

IP地址怎么来呢？打开终端，输入ping 用户名.github.io

四、我的记录类型显示的是两个A，然后做了修改，最终效果如下：

五、在博客根目录source中新建CNAME记事本，内容输入www.flipped97.me【个人域名】，保存关闭。

然后重命名，删除后缀.txt，效果如下：

ps：如果看不到文件的后缀名.txt，工具–文件夹选项–查看–取消下图所示的勾选即可。

六、进入github设置，绑定域名



七、浏览器输入www.flipped97.me，成功打开，完成！

ps：浏览器输入flipped-97.github.io会自动跳转到www.flipped97.me
