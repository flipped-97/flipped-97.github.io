CODING中找不到Pages服务

2020-07-28
-
coding

近期在捣鼓个人博客，使用基于Hexo博客框架以及Github Pages+个人域名的访问方式，但是访问速度并不理想。故采用github+CODING双线部署的方式！

创建项目并配置好SSH公钥后，代码仓库中并没有Pages服务 解决方法：点击左侧导航栏项目设置，在项目与成员->功能开关中打开构建与部属功能
